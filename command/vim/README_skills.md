# commons
- cw    - 删除从光标位置到单词结尾之间的字符并进入插入模式
- ciw   - 删除光标所在单词并进入插入模式
- daw   - 删除光标所在位置的单词及空格
- vit   - 高亮选中标签内部的内容
- w     - 正向移动到下一单词的开头
- b     - 反向移动到当前单词/上一单词的开头
- n|    - 移动到当前行的第n列
- zz    - 重绘屏幕并把当前行显示在窗口正中
- zt    - 将光标所在行移到窗口的顶端
- zb    - 将光标所在行移到窗口的底部
- <ctrl-i>    - 跳转位置后退
- <ctrl-0>    - 类似浏览器的前进后退功能
- ''    - 回到上次跳转的位置
- '.    - 回到上次编辑的位置

# Mark
@:      - 重复执行上次的Vim命令行模式命令 
dG/:%d  - 刪除所有行
xp      - 调换Vim当前光标之后的两个字符的顺序
f{char} - 正向移动到下一个 {char} 所在之处的字符上
t{char} - 正向移动到下一个 {char} 所在之处的前一个字符上
